

Este repositório armazena dashboards que criei com os conhecimentos obtidos no curso
Microsoft Power BI Para Business Intelligence e Data Science disponibilizado pela Data Science Academy,em sua plataforma. Cada arquivo possui os datasets utilizados para gerar as analises e os relatórios elaborados (Arquivos .pbix). Cada pasta neste diretório possui um arquivo de texto que contextualiza para qual área de negocio e os indicadores que serão explorados o relatorio (Arquivos .txt), e os arquvios contendo os datasets utilizados para gerar as analises (Arquivos .xlsx e .csv)

Todos os dados foram obtidos atravé das plataforma da D.S.A: https://www.datascienceacademy.com.br/path-player?courseid=microsoft-power-bi-para-business-intelligence-e-data-science\&unit=63ca400485dcf67e2200bb4bUnit
